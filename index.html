SEÇÃO 02 — A TECNOLOGIA

Como detectamos o fogo

Combinamos dados orbitais abertos, sensores IoT e análise em Python para transformar imagens termais em alertas acionáveis.

🛸

NASA FIRMS

Imagens dos satélites MODIS e VIIRS com resolução de 375 metros e atualização a cada passagem orbital.

Dados orbitais
🔌

Arduino IoT

Sensores de temperatura (LM35) e fumaça (MQ-2) em campo. Complementam os dados satelitais com leituras locais.

Edge Computing
🐍

Python Analytics

Modelagem de propagação usando funções exponenciais. Prevê área afetada com base no vento e umidade.

Análise preditiva
📡

API de Alertas

Pipeline que processa detecções e dispara notificações para autoridades e moradores da área de risco.

Automação
🗺️

Dashboard Geoespacial

Mapa interativo com focos ativos, histórico regional e camadas de risco sobrepostas a dados climáticos.

Visualização
☁️

INPE Open Data

Banco histórico de focos do INPE — base para padrões sazonais e validação das detecções.

Dados históricos

SEÇÃO 03 — OBJETIVOS

O que o FireWatch entrega

Cada objetivo foi pensado para reduzir o tempo entre a detecção e a resposta.

01

Detecção precoce automatizada

Identificar focos em até 3 minutos após nova imagem orbital disponível.

02

Alertas geolocalizados

Notificar usuários cadastrados por proximidade ao foco ativo.

03

Predição de propagação

Estimar área em risco nas próximas horas com modelagem matemática.

04

Relatórios para gestores

Painel de análise histórica por bioma, estado e período.

SEÇÃO 04 — PÚBLICO-ALVO

Quem usa o FireWatch

Do cidadão em área de risco ao gestor público que precisa de dados confiáveis para agir com rapidez.

🚒

Bombeiros

Recebem coordenadas do foco e estimativa de propagação para planejar o combate.

🌿

IBAMA & INPE

Acesso ao histórico analítico para políticas públicas e relatórios de desmatamento.

🧑‍🌾

Produtores Rurais

Alertas preventivos para proteger propriedade, colheita e rebanho a tempo.

🏘️

Moradores de Risco

Notificação no celular quando há foco ativo no raio configurado da localização.
